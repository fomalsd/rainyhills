## Rainy Hills
CRX Markets test task

### Technologies used
* Java 11
* Spring Boot
* JUnit
* SLF4J

### Complexity
Time: O(n log(n)) \
Memory: O(n)

### How to run
#### Requirements:
* JDK 11
* Maven

### Execution Commands
    mvn clean package spring-boot:run 
    	
Visit the main page at http://localhost:8080/ after the service has started.
